# AI Expense & Bills Agent (Gemini, Advanced Edition)

This project implements an AI Agent for personal expense & bills management.

- Uses **Google Gemini** as a planner (LLM integration).
- Converts natural language into structured **JSON action plans**.
- Executes actions on:
  - SQLite database (`expenses` and `bills`),
  - Local filesystem (Markdown reports, JSON logs),
  - Analytical computations (savings goal planning & spending health check).
- Provides both a **terminal UI** and a **Streamlit web UI** suitable for publishing.
